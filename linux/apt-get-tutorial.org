* apt-get意外中断解决办法
 Ctrl+c/z中断后
 中断后会现实已经锁定,并且用fg和bg都无法恢复
 用sudo kill -9 apt-get也无法关闭的时候
 
 解决方法
 #+BEGIN_SRC shell
 $sudo killall apt-get
 $sudo killall dpkg
 $sudo dpkg --configure -a
 #就完美的搞定了
 #+END_SRC

* dpkg安装和卸载软件教程
#+BEGIN_SRC shell
$sudo dpkg -i ***.deb #安装
$sudo apt-get install -f #一般会出现依赖关系,直接这样安装即可
$sudo dpkg -l  #查看所有的软件包名称
$sudo dpkg --remove ***  #卸载软件包
#+END_SRC
