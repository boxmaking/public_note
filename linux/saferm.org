* 命令行实现回收站功能
1. 编写remove.sh文件
   #+INCLUDE: "/home/pengshao/shellscript/remove.sh"
2. 编写~/.bashrc文件
   #+BEGIN_SRC sh
   $alias rm="sh ~/shellscript/remove.sh"
   #+END_SRC
3. 加载文件
   #+BEGIN_SRC sh
   $source ~/.bashrc   
   #+END_SRC
4. 定期清理
   #+BEGIN_SRC sh
   0 0 * * * rm -rf /home/.trash/* 
   #+END_SRC
